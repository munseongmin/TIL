# Linux Command
> 리눅스 명령어들을 정리해 봅시다.

- pwd(print working directory)
    - 현재 작업중인 경로를 출력
- ls(list)
    - 현재 폴더에 있는 파일, 폴더를 출력
    - `-a`(optional) : 숨김 처리된 파일, 폴더까지 출력
- cd(change directory)
    - 폴더 이동
- mkdir(make directory)
    - 폴더 생성
- touch
    - 파일 생성
- rm(remove)
    - 파일 및 폴더 삭제
    - `-r`(optional) : 폴더를 지우기 위해서 입력하는 옵션
